# 🌸 Bloom || Flowers Studio

אתר חנות פרחים בוטיק – פרויקט קורס פיתוח אלגוריתם  
נבנה ב-HTML · CSS · Vanilla JavaScript

---

## תיאור הפרויקט

**Bloom** היא חנות פרחים עדינה ואלגנטית המאפשרת ללקוחות לעיין בקולקציית זרים, להוסיף מוצרים לסל קניות, לבצע הזמנה עם פרטי משלוח וליצור קשר עם החנות.

האתר בנוי ב-HTML, CSS ו-JavaScript טהור (Vanilla JS) ללא frameworks חיצוניים, עם עיצוב רספונסיבי (Mobile + Desktop) ושמירה על שפה עיצובית עדינה ונשית.

## עמודי האתר – תיעוד מלא

### עמוד 1 – דף הבית (`index.html`)

**User Story**
> כמבקרת חדשה באתר, אני רוצה להבין מיד מה האתר מציע ולהרגיש את האווירה של המותג, כדי שארצה להיכנס ולראות את הקולקציה.

**Elements – רכיבים בעמוד**
- Header: לוגו Bloom + קישורי ניווט + אייקון עגלה עם badge
- Hero Image – תמונת פתיחה מיתוגית
- כותרת: *"You're about to bloom"*
- פסקת תיאור המותג בעברית
- כפתור CTA ראשי
- Footer: טופס הצטרפות לניוזלטר + זכויות יוצרים

**Hierarchy – סדר חשיבות ויזואלי**
1. Logo + Navbar (אוריינטציה – איפה אני?)
2. תמונת Hero (רגש + מיתוג)
3. כותרת + תיאור (מה המקום הזה?)
4. כפתור CTA (מה לעשות?)
5. Newsletter Footer (הצטרפות קהילה)

**CTA**
> **"צפי בקולקציה שלנו"** → מנווט ל-`flowers.html`

---

### עמוד 2 – הזרים שלנו (`flowers.html`)

**User Story**
> כלקוחה שרוצה לקנות פרחים, אני רוצה לראות את כל הזרים הזמינים עם תמונות, שמות ומחירים, ולהוסיף לסל את מה שמצא חן בעיניי בלחיצה אחת.

**Elements – רכיבים בעמוד**
- Header: לוגו + ניווט + cart badge (מעודכן real-time)
- כותרת עמוד + תיאור הקולקציה
- רשת מוצרים (CSS Grid, responsive): 7 כרטיסי זר
  - כל כרטיס: תמונה, שם הזר, מחיר, כפתור "הוספה לסל"
- Toast notification (תחתית המסך) בעת הוספה לסל
- Footer: ניוזלטר + copyright

**Hierarchy – סדר חשיבות ויזואלי**
1. כותרת העמוד (אוריינטציה)
2. תמונת המוצר (מניע רכישה)
3. שם הזר + מחיר
4. כפתור הוספה לסל
5. Cart Badge (feedback מיידי)

**CTA**
> **"הוספה לסל"** על כל כרטיס מוצר → מעדכן localStorage + badge

---

### עמוד 3 – עגלת קניות (`cart.html`)

**User Story**
> כלקוחה שבחרה מוצרים, אני רוצה לראות את כל הפריטים בסל שלי, לשנות כמויות או למחוק, לראות את הסכום הכולל ולבצע הזמנה עם פרטי משלוח.

**Elements – רכיבים בעמוד**
- Header: ניווט + cart badge
- כותרת "הפריטים שלך"
- רשימת פריטי הסל (נוצרת דינמית מ-localStorage):
  - תמונה + שם + מחיר × כמות
  - כפתורי +/−/🗑 לכל פריט
- סכום כולל (מתעדכן בזמן אמת)
- מצב ריק: הודעה מנחה חוזרת ל-flowers.html
- טופס הזמנה (מוסתר כשהסל ריק):
  - שם פרטי, שם משפחה, כתובת, אימייל, תאריך משלוח
- כפתור "בצע הזמנה"
- Footer: ניוזלטר

**Hierarchy – סדר חשיבות ויזואלי**
1. פריטי הסל (תוצר הבחירה)
2. מחיר × כמות + כפתורי שינוי
3. סכום כולל (החלטה: להמשיך?)
4. טופס פרטי הזמנה
5. כפתור ביצוע הזמנה

**CTA**
> **"בצע הזמנה"** → מנקה סל + מנווט ל-`success.html`

---

### עמוד 4 – צור קשר (`contact.html`)

**User Story**
> כלקוחה שיש לה שאלה על מוצר או הזמנה מיוחדת, אני רוצה למצוא בקלות את פרטי הקשר של החנות ולשלוח הודעה ישירה דרך האתר.

**Elements – רכיבים בעמוד**
- Header: ניווט + cart badge
- כותרת עמוד + תת-כותרת
- תיבת פרטי התקשרות: כתובת, טלפון, אימייל, שעות פתיחה
- טופס פנייה: שם מלא, אימייל, הודעה (textarea), כפתור שליחה
- Footer: ניוזלטר

**Hierarchy – סדר חשיבות ויזואלי**
1. כותרת (אוריינטציה)
2. פרטי התקשרות (מידע מהיר)
3. טופס שליחת הודעה (פעולה)
4. Newsletter

**CTA**
> **"שליחה"** (טופס יצירת קשר) → מנווט ל-`thankyou.html`

## פיצ'רים עיקריים

- **עגלת קניות** – localStorage לשמירת פריטים בין דפים
- **Cart Badge** – מספר פריטים מוצג על אייקון הסל בכל הדפים, מתעדכן real-time
- **Toast Notification** – הודעת הצלחה בתחתית המסך בעת הוספה לסל
- **ניוזלטר** – טופס הצטרפות עם הודעת אישור מונפשת
- **Responsive Design** – תמיכה ב-Desktop, Tablet, Mobile
- **RTL Support** – כל האתר בכיוון ימין לשמאל (עברית)
