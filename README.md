# Bloom

Bloom הוא אתר שבניתי לחנות פרחים.

באתר אפשר לראות סוגים שונים של פרחים, לסנן מוצרים, להוסיף פרחים לסל, לשנות כמויות ולהסיר מוצרים מהסל.

הסל נשמר ב LocalStorage, ולכן הוא לא נמחק כשמרעננים את הדף.

## איך מפעילים את האתר

פותחים את תיקיית הפרויקט ב VS Code.

אחר כך פותחים את הקובץ index.html דרך Live Server.

## מאיפה מגיעים הנתונים

הפרחים והמחירים שלהם נמצאים בקוד של הפרויקט.

המידע של סל הקניות נשמר בדפדפן באמצעות LocalStorage.

## מגבלות

באתר אין תשלום אמיתי.

אין הרשמה של משתמשים ואין חיבור למסד נתונים.
