# PRD – Bloom

## מטרת הפרויקט


האתר מיועד לאנשים שרוצים לקנות פרחים לאירועים כמתנה או סתם בשביל הכיף.

## משתמשי יעד


המטרה היא לאפשר להם למצוא פרחים שמתאימים להם בקלות ולהעניק חוויית משתמש נוחה, נעימה ופשוטה.

## מה כלול בפרויקט

- דף בית
- קטלוג פרחים
- חיפוש וסינון מוצרים
- הוספת מוצרים לסל
- שינוי כמות של מוצרים בסל
- הסרת מוצרים מהסל
- חישוב מחיר כולל
- שמירת הסל בLocalStorage
- טופס יצירת קשר
- עיצוב שמתאים גם למסכים קטנים

## מה לא כלול בפרויקט

- תשלום אמיתי
- הרשמה והתחברות של משתמשים
- חיבור למסד נתונים
- ניהול מלאי אמיתי
- מערכת ניהול לבעל החנות

## דרישות פונקציונליות

- המשתמש יכול לצפות ברשימת הפרחים
- המשתמש יכול לחפש או לסנן פרחים
- המשתמש יכול להוסיף מוצר לסל
- המשתמש יכול לשנות כמות של מוצר
- המשתמש יכול להסיר מוצר מהסל
- האתר מחשב את המחיר הכולל
- הסל נשמר גם אחרי רענון הדף
- המשתמש יכול לשלוח טופס יצירת קשר

## בחירות טכניות

הפרויקט נבנה באמצעות:

- HTML
- CSS
- JavaScript
- LocalStorage

## נתונים

נתוני הפרחים והמחירים נמצאים בתוך הקוד של הפרויקט.

נתוני סל הקניות נשמרים בדפדפן באמצעות LocalStorage.

## הגדרת סיום הפרויקט

הפרויקט נחשב מוכן להגשה לאחר שכל הפונקציות המרכזיות נבדקו ונמצאו תקינות

- מעבר תקין בין כל העמודים
- חיפוש וסינון פרחים
- הוספה, הסרה ושינוי כמויות בסל
- חישוב נכון של המחיר הכולל
- שמירת הסל לאחר רענון הדף
- תצוגה תקינה במחשב ובמובייל
